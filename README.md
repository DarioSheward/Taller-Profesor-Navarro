Este repositorio acoje algunos scripts dedicados a utilizar el HMF (Hamiltonian Mean-field Model), estudiando casos de estado cuasiestacionario. 

Al tratarse de scripts premiliminares, los párametros establecidos no buscan ser representativos de los sistemas a estudiar, sino que buscan agilizar el proceso de retroalimentacion y escritura.

Para revisar los scripts gráficos revise [esta carpeta](https://github.com/DarioSheward/Taller-Profesor-Navarro/tree/main/scripts-dedicados). Basta con correr simulacion.py y video.py para obtener un video. Aliento a cambiar párametros. Pero aún queda bastante por optimizar.
